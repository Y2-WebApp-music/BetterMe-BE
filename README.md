# BetterMe Backend

### Install Bun:
* Windows 
```bash
powershell -c "irm bun.sh/install.ps1 | iex"
```
* Linux & macOS
```bash
curl -fsSL https://bun.sh/install | bash
```


## Development
To start the development server run:
```bash
bun run dev
```

Open http://localhost:3000/ with your browser to see the result.

## Docker
Build :
```bash
docker compose build
```

Run :
```bash
docker compose up -d
```
